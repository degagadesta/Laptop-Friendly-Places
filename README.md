# Laptop Friendly Places (LFP)

Laptop Friendly Places (LFP) is a web platform that helps students, freelancers, and remote workers find places that are suitable for working with a laptop. These places usually provide reliable Wi-Fi, a quiet environment, power outlets, and good customer service.

The platform allows users to explore locations such as cafés, libraries, and coworking spaces that provide a comfortable environment for studying or working.

## Project Goal

The main goal of this project is to create a system where users can easily discover and share laptop-friendly places in their area.

Users will be able to:
- Browse laptop-friendly places
- View detailed information about each place
- See ratings for Wi-Fi, power availability, and customer service
- Save favorite places
- Explore places on an interactive map
- Contribute new places
- Report incorrect or inappropriate place information

## Technology Stack

**Frontend**
- HTML5
- CSS3
- JavaScript (ES6)

**Backend**
- PHP (for handling server-side logic and APIs)

**Database**
- MySQL (planned for storing users, places, reports, and ratings)

**Map Integration**
- Leaflet.js with OpenStreetMap
